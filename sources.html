Ресурсы Lampa

Официальные каналы Lampa

  • Lampa(@lampa_group) — Основной Telegram-чат для обсуждений и инструкций по Lampa.
  • Lampa - Аварийная(@lampa_plugins) — Telegram-канал с актуальными плагинами.
    • Закреп #2 — Список актуальных плагинов.
    • Сообщение #285026 — Альтернативный адрес для плагина Онлайн Мод.
  • Lampa - Аварийная(@lampa_unavailable) — Telegram-канал для решения аварийных случаев.
    • Сообщение #4721 — Инструкции по устранению проблем с пустым списком.

Смежные каналы

  • TorrServe MatriX(@TorrServe) — Официальный чат по TorrServe для настройки и поддержки.
  • Телевидение в LAMPA(@lampaiptv) — Telegram-чат для IPTV-плейлистов, включая украинские каналы.
    • Сообщение #30886 — Инструкции по настройке украинских телеканалов.
  • Чат Showy — Поддержка плагина для просмотра видеоконтента Showy.
  • Чат MODSS — Поддержка плагина для просмотра видеоконтента MODSS .
